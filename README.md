# NeuroNest - Smart Note Taking Application

A professional and fully responsive post-it notes application built with Vue.js 3, Vite, Pinia, and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
npm run build
```

Visit `http://localhost:5173` to use the application.

## Features

- Create, edit, and delete notes
- Auto-save notes to browser storage
- Responsive design for all devices
- Fast performance with code splitting
- Clean and modern interface
- Accessible design
- Notes sorted by recent updates

## Project Structure

```
src/
├── views/              # Page components
├── components/         # Reusable UI components
├── composables/        # Vue 3 composition logic
├── stores/            # Pinia state management
├── utils/             # Helper utilities
├── router/            # Vue Router configuration
├── types/             # Type definitions
├── constants/         # App constants
└── assets/            # Styles and static files
```

## Technology Stack

- Vue.js 3 - Progressive JavaScript framework
- Vite - Next generation build tool
- Pinia - State management
- Vue Router - Client-side routing
- Tailwind CSS 4 - Utility-first CSS

## Available Scripts

```bash
npm run dev          # Start development server
npm run build        # Build for production

```

## Development

The project uses Vue.js 3 Composition API with composables for logic reuse and Pinia for state management.
